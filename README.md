# Fungal Nutrient Analysis under Environmental Stress

## Overview
This project simulates and analyzes the nutrient uptake (specifically Phosphorus and Potassium) of various fungal strains under varying environmental stress conditions. The analysis bridges biological concepts with computational data processing to identify the most physiologically resilient strains under drought stress.

## Technologies Used
* **Python 3**
* **Pandas:** Data structuring, cleaning, and manipulation.
* **NumPy:** Statistical analysis and biological data simulation.
* **Jupyter Notebook:** Interactive data visualization and step-by-step execution.

## Project Workflow
1. **Data Generation:** Simulated a realistic dataset of 500 fungal samples across three distinct strains, injecting natural biological variance.
2. **Data Cleaning:** Identified and filtered out corrupted/missing simulated lab data using Pandas.
3. **Statistical Analysis:** Extracted mean nutrient uptake metrics to determine which fungal structure maintained the highest physiological resilience during drought conditions.
